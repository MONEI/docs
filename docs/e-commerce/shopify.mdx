To accept payments through MONEI in your Shopify store you simply need to install and configure MONEI Payments Shopify App.

<p>
<a href="https://monei-demo.myshopify.com/" target="_blank" className="button button--primary">
  Live demo
</a>
</p>

Password: `demo`

## Before you begin

To test your integration:

- Use your [test mode](testing.md) API Key. You can find them in [MONEI Dashboard → Settings → API Access](https://dashboard.monei.com/settings/api)
- You can check the status of a test payment in your [MONEI Dashboard → Payments](https://dashboard.monei.com/payments) (in test mode).

## Install

<p>
  <a href="https://apps.shopify.com/monei-payments" target="_blank" className="button button--primary">
    Install from Shopify App Store
  </a>
</p>

## Configure

Go to **Shopify Admin → Apps → MONEI Payments**

Enter your API Key. You can find your API Key in [MONEI Dashboard → Settings → API Access](https://dashboard.monei.com/settings/api).

![Configure MONEI Shopify](/img/configure-shopify-1.png)

After you save your API Key you will be redirected to the Payments Settings. Click on **Activate MONEI Payments**.

![Configure MONEI Payments](/img/configure-shopify-2.png)

## Before you go live

- Make sure that you are using [live (production) mode](testing.md) API Key.
- Make sure that you have at least one active [payment method](https://dashboard.monei.com/settings/payment-methods)

## How to find the Shopify Order

To find the order in Shopify Admin use the Order ID from MONEI Dashboard

![Find MONEI Payment](/img/configure-shopify-4.png)

Enter the following query in Shopify Admin -> Orders

`receipt.payment_id:{{ MONEI_ORDER_ID }}`

![Find MONEI Payment in Shopify](/img/configure-shopify-3.png)

You can find the MONEI Order ID in the **Information from the gateway** section in the Shopify order.

![Find Payment in MONEI](/img/configure-shopify-5.png)
