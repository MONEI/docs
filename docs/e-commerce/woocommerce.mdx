To accept payments through MONEI in your WooCommerce store you simply need to install and configure MONEI WooCommerce Plugin.

## Before you begin

To test your integration:

- Use your [test mode](testing.md) Account ID and Password. You can find them in [MONEI Dashboard → Settings](https://dashboard.monei.com/settings)
- You can check the status of a test payment in your [MONEI Dashboard → Payments](https://dashboard.monei.com/payments) (in test mode).

## Install

Search for **MONEI WooCommerce** Plugin in your WordPress site or download it from official WordPress plugins page.

<p>
  <a href="https://wordpress.org/plugins/monei/" target="_blank" className="button button--primary">
    Download
  </a>
</p>

## Configure

Enter your API Key. You can find your API Key in [MONEI Dashboard → Settings → API Access](https://dashboard.monei.com/settings/api).

## Before you go live

- Make sure that you are using [live (production) mode](testing.md) API Key.
- Make sure that you have at least one active [payment method](https://dashboard.monei.com/settings/payment-methods)
