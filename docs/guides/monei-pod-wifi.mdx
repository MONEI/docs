Follow these steps to configure WiFi on your MONEI POD:

<Tabs
  defaultValue="android"
  values={[
    {label: 'Android', value: 'android'},
    {label: 'iOS', value: 'ios'}
  ]}>
<TabItem value="android">

1. Install [MFAPLink](https://play.google.com/store/apps/details?id=com.morefun.aplinkdemo) on your phone.

2. Long press the **M** key for 2 seconds on your MONEI POD to enter **Configure Network Mode**.

   <img
     src="/img/monei-pod-wifi/monei-pod-1.jpg"
     alt="MONEI POD Configure Network Mode screen"
     width="300"
   />

3. Press **+** to select **WIFI** and enter WiFi configuration mode.

   <img
     src="/img/monei-pod-wifi/monei-pod-2.jpg"
     alt="MONEI POD WIFI Network Mode selection"
     width="300"
   />

4. Connect your phone to the WiFi network you want to configure on the MONEI POD (must be 2.4GHz, 5GHz is not supported).

5. Open MFAPLink app and select **KM_AP CONNECTION**.

   <img
     src="/img/monei-pod-wifi/android-1.jpg"
     alt="MFAPLink Android app KM_AP CONNECTION option"
     width="300"
   />

6. Verify the WiFi name is correct, enter the WiFi password, and tap **CONNECT**.

   <img
     src="/img/monei-pod-wifi/android-2.jpg"
     alt="MFAPLink Android WiFi configuration screen"
     width="300"
   />

7. Allow MFAPLink to join the MONEI POD's network when prompted.

   <img
     src="/img/monei-pod-wifi/android-3.jpg"
     alt="Android permission dialog to join MONEI POD network"
     width="300"
   />

8. Wait for the success message: **Ap network configuration information is sent successfully**. The device will restart and connect to the WiFi network.
   <img
     src="/img/monei-pod-wifi/android-4.jpg"
     alt="MFAPLink Android success message confirmation"
     width="300"
   />

</TabItem>

<TabItem value="ios">

1. Install [MFAPLink](https://apps.apple.com/es/app/mfaplink/id1645497573) on your phone.

2. Long press the **M** key for 2 seconds on your MONEI POD to enter **Configure Network Mode**.

   <img
     src="/img/monei-pod-wifi/monei-pod-1.jpg"
     alt="MONEI POD Configure Network Mode screen"
     width="300"
   />

3. Press **+** to select **WIFI** and enter WiFi configuration mode.

   <img
     src="/img/monei-pod-wifi/monei-pod-2.jpg"
     alt="MONEI POD WIFI Network Mode selection"
     width="300"
   />

4. Connect your phone to the WiFi network you want to configure on the MONEI POD (must be 2.4GHz, 5GHz is not supported).

5. Open MFAPLink app and select **Default AP Mode**.

6. Verify the WiFi name is correct, enter the WiFi password, and tap **Connect**.

   <img
     src="/img/monei-pod-wifi/iphone-1.jpg"
     alt="MFAPLink iOS WiFi configuration screen"
     width="300"
   />

7. Allow MFAPLink to join the MONEI POD's network when prompted.

   <img
     src="/img/monei-pod-wifi/iphone-2.jpg"
     alt="iOS permission dialog to join MONEI POD network"
     width="300"
   />

8. Wait for the success message: **Connecting device successfully**. The device will restart and connect to the WiFi network.
   <img
     src="/img/monei-pod-wifi/iphone-3.jpg"
     alt="MFAPLink iOS success message confirmation"
     width="300"
   />

</TabItem>
</Tabs>
