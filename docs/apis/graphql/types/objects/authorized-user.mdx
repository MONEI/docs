An authorized user that can access the MONEI dashboard.
One user can have access to multiple accounts with different access levels.

```graphql
type AuthorizedUser {
  userEmail: String!
  id: ID!
  accessLevel: AuthorizedUserAccessLevel!
  storeId: ID
  status: AuthorizedUserStatus!
  lastLoginAt: String
  createdAt: String!
  updatedAt: String
}
```
