<p>Retrieve available payment methods for an account or a specific payment.</p>
<p>
  You can provide either an <code>accountId</code> or a <code>paymentId</code> as a query parameter
  to get the available payment methods.
</p>
<p>
  When providing a <code>paymentId</code>, the response will include additional payment-specific
  information such as amount and currency. This is useful when you want to show payment options that
  are specifically available for a particular transaction.
</p>
<p>
  When providing an <code>accountId</code>, the response will include all payment methods available
  for that account based on the merchant&#39;s configuration and supported payment methods.
</p>
