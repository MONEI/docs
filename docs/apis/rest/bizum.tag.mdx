<p>The Bizum API allows you to integrate with Spain&#39;s popular mobile payment system.</p>
<p>With the Bizum API, you can:</p>
<ul>
  <li>Validate customer phone numbers for Bizum compatibility</li>
  <li>Process payments using Bizum as a payment method</li>
  <li>Offer a seamless mobile payment experience for Spanish customers</li>
</ul>
<p>
  Bizum is a widely used payment method in Spain that allows customers to make payments using their
  mobile phone number linked to their bank account. This API provides the necessary tools to
  incorporate Bizum into your payment options.
</p>
<h2>Bizum validation</h2>
<p>
  Before processing a Bizum payment, you can validate if a customer&#39;s phone number is registered
  with Bizum using the validate-phone endpoint.
</p>
