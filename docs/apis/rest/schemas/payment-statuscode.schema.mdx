<Heading as={'h1'} className={'openapi__heading'} children={'Payment-StatusCode'}></Heading>

<p>
  The MONEI API returns a <code>statusCode</code> attribute in the{' '}
  <a href="https://docs.monei.com/docs/apis/rest/payments/">Payment Object</a> to indicate the
  outcome of your request. This helps you understand whether an operation was successful or why it
  failed.
</p>
<h3>Success Code</h3>
<p>Successful transactions return the following code:</p>
<table>
  <thead>
    <tr>
      <th>Code</th>
      <th>Message</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>E000</td>
      <td>Transaction approved</td>
    </tr>
    <tr>
      <td>E001</td>
      <td>Transaction pending processing</td>
    </tr>
    <tr>
      <td>E002</td>
      <td>Transaction pending authentication</td>
    </tr>
    <tr>
      <td>E003</td>
      <td>Transaction pending authorization</td>
    </tr>
  </tbody>
</table>
<h3>Error Codes</h3>
<p>Error codes are grouped by category to help you quickly identify the type of issue:</p>

<p>These errors relate to account configuration issues:</p>

      <th>Message</th>
    </tr>
  </thead>

      </td>

      <td>E102</td>
      <td>Invalid or inactive MID. Please contact the acquiring entity</td>
    </tr>
    <tr>
